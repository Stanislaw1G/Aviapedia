# Aviapedia
Aviapedia OS to zaawansowane centrum informacyjne poświęcone lotnictwu cywilnemu. Projekt został stworzony jako interaktywna baza wiedzy, oferująca szybki wgląd w specyfikacje techniczne, historię konstrukcji oraz kluczowe dane operacyjne najpopularniejszych samolotów pasażerskich świata.
